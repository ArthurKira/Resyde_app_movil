# Assets - Imágenes

## Logo de la aplicación

Coloca el archivo del logo de UCLUB GROUP en esta carpeta con el nombre:

**`logo.png`** o **`logo.jpg`**

El logo se mostrará automáticamente en la página de login.

### Formatos soportados:
- PNG (recomendado)
- JPG/JPEG

### Tamaño recomendado:
- Ancho: 300-400px
- Alto: proporcional
- Resolución: 72-150 DPI (para pantallas)
